# spin-wheel
抽獎旋轉盤，參考: https://mirari.cc/2019/06/25/CSS3实现转盘抽奖/

**使用 Vue CLI 建立的專案，但主要是裡面的 src/components/SpinWheel.vue**

## props 
| 型態    | 說明            | 範例                    |
|-------- |---------------- |------------------------ |
| Array   | 獎項列表        | [{icon: "", name: ""}]  |
| Number  | 旋轉動畫的時間   | 5 (5秒)                  |
